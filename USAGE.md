# 사용법

## 1. 데이터 갱신 스크립트

### 1-1. 매일 갱신 (자동 — GitHub Action)
```bash
python update_rsi.py
```
- 일/주 RSI, 시세, EPS·매출 YoY, 시장 위젯 데이터를 `data/*.json`에 저장
- `data/backtest.json` 캐시가 있으면 자동 읽어서 백테스트 통계를 메인 데이터에 머지

### 1-2. 종목 상세 페이지 생성
```bash
python scripts/generate_financials.py
```
- `stocks/{ticker}.html` + `stocks/{ticker}_chart.json` 생성
- 재무 차트 + 일/주 RSI+가격 듀얼축 차트 포함

### 1-3. 백테스트 (주 1회 / 수동 실행) ⭐ 신규
```bash
python scripts/backtest.py
```
- 5년치 일봉으로 RSI≤30 진입 시점 자동 탐지 → 1M/3M/6M 후 수익률·승률 계산
- 결과는 `data/backtest.json`에 캐시
- **무거움** (200종목 × 5년치, 약 10~20분 추정)
- 매일 돌릴 필요 없음 — 주 1회 또는 신규 종목 추가 시만 실행
- 실행 후 `python update_rsi.py` 한 번 더 돌리면 메인 테이블에 통계 반영됨

### 실행 순서 (전체 갱신 시)
```bash
python scripts/backtest.py          # (주 1회) 백테스트 캐시 갱신
python update_rsi.py                # (매일) 시세·RSI·백테스트 통계 머지
python scripts/generate_financials.py  # (매일) 상세 페이지 갱신
```
`update_rsi.py`와 `generate_financials.py`는 서로 독립 — 순서 무관.

## 2. 메인 페이지 컬럼 해석

### RSI 관련
| 컬럼 | 의미 |
|---|---|
| 일 RSI | 일봉 14일 RSI (Wilder EMA) |
| 일 RSI 30일 추세 | 최근 30거래일 일봉 RSI 스파크라인 (30 이하 파랑 점, 70 이상 빨강 점) |
| 주 RSI | 주봉(W-FRI 리샘플) 14주 RSI |
| 주 RSI 30주 추세 | 최근 30주 주봉 RSI 스파크라인 |
| 🎯 쌍과매도 | 일 RSI ≤ 35 AND 주 RSI ≤ 35 — 다중 시간프레임 과매도 신호 |

### 시세
| 컬럼 | 의미 |
|---|---|
| 현재가 | 최근 종가 |
| 일변동(%) | 전일 대비 |
| 52주 위치(%) | (현재가 - 52주 저가) / (52주 고가 - 52주 저가). 0이면 저점, 100이면 고점 |
| 거래량 비율 | 최근 거래량 / 20일 평균. 1.5x 이상 = 평소보다 많음 |

### 우상향 지표 (📈)
| 컬럼 | 의미 |
|---|---|
| EPS 성장(%) | (Forward EPS - Trailing EPS) / |Trailing EPS| × 100. 시장의 1년 성장 기대치 |
| 매출 YoY(%) | 최근 분기 매출 / 작년 동기 매출 - 1 × 100. 실제 펀더멘털 추세 |

**해석**: 두 지표 모두 양수 → 우상향. 둘 다 음수 → 회피.

### 백테스트 (🧪) — v10.12 신규
| 컬럼 | 형식 | 의미 |
|---|---|---|
| 🧪 1M 평균/승률 | `+5.2% / 65% (N=12)` | 과거 5년 RSI≤30 진입 후 1개월 후 평균 수익률 / 승률 / 진입 사건 수 |
| 🧪 3M 평균/승률 | `+12.1% / 72% (N=12)` | 3개월 보유 시 통계 |

**색상 해석**:
- 🟩 진한 초록 = 평균 +5% 이상 & 승률 70%+ → **강한 매수 신호**
- 🟢 옅은 초록 = 평균 +3% 이상 & 승률 60%+ → 양호
- 🟥 빨강 = 평균 음수 → **회피 권장** (RSI 30 와도 평균적으로 손실)
- N (진입 횟수)이 너무 적으면 (예: 3건 미만) 통계 신뢰도 낮음 — 참고만

## 3. 필터 칩 사용법

| 칩 | 동작 |
|---|---|
| 전체 | 필터 해제 |
| 일 RSI ≤ 30 | 일봉 과매도 종목만 |
| 일 RSI ≤ 35 | 일봉 약한 과매도 포함 |
| 주 RSI ≤ 35 | 주봉 과매도 |
| 🎯 일·주 동시 ≤ 35 | **다중 시간프레임 과매도** — 가장 신뢰도 높은 신호 |
| 최근 7일 과매도 | 최근 일주일 내 RSI≤30 진입 이력 |

**섹터 드롭다운**과 **검색바**(티커/종목명)는 위 필터와 AND 조합으로 동작.

## 4. 시장 벤치마크 위젯

상단 4개 카드 (SPY/QQQ/DIA/IWM) — 매수 전 시장 컨텍스트 확인용.
- 시장 RSI도 일/주 동시 표시
- 시장 전체 과매도면 개별 신호 신뢰도 ↑
- 시장 상승 중 개별 종목만 과매도면 → 개별 이슈 의심

---

## 5. 종목 상세 페이지 (Stock Detail)

티커 또는 종목명 클릭 시 이동.

- **일봉 RSI + 가격 듀얼축 차트 (6개월)** — 단기 진입 타이밍 확인
- **주봉 RSI + 가격 듀얼축 차트 (2년)** — 장기 추세 확인, 쌍돌파 시점 시각화
- 분기/연간 재무 표 + Plotly 매출·이익·FCF 차트
- "← 목록으로" 링크로 뒤로가기

## 6. 트러블슈팅

| 증상 | 해결 |
|---|---|
| 🧪 백테스트 컬럼이 비어 있음 | `python scripts/backtest.py` 한 번 실행 |
| 주봉 RSI가 없음 | `data/*.json`이 v10.11 이전 데이터 — `python update_rsi.py` 재실행 |
| 시장 위젯이 "데이터 없음" | `python update_rsi.py`가 `data/market.json` 생성하는지 확인 |
| 종목 상세 RSI 차트 없음 | `python scripts/generate_financials.py` 재실행 (v10.11+ 필요) |
